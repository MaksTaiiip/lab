# lab
Блок-пазл — це інтелектуальна гра на базі Pygame, де гравець розміщує фігури на сітці 10×10, очищує лінії та змагається за максимальну кількість балів у трьох різних режимах.

Мова програмування: Python
Бібліотека графіки: Pygame
Додаткові модулі: copy, random, base64, tempfile


Інструкція з використанням
1.Встановіть Python
2.Встановіть бібліотеку Pygame через термінал: pip install pygame
3.Завантажте проєкт або склонуйте репозиторій: git clone https://github.com/MaksTaiiip/lab.git
4.Запустіть гру: python main.py


Режими гри та керування
1.Соло (Single Player)
Класичний режим на виживання та набір очок.
Перетягуйте 3 випадкові фігури на поле за допомогою миші.
Заповнюйте вертикальні або горизонтальні лінії для їх зникнення (+100 очок).
За кожну поставлену фігуру нараховується +30 очок.
Складність зростає разом із вашим рахунком.

2.Локальний мультиплеєр (PVP)
Гра для двох людей за одним комп'ютером.
Оберіть ліміт фігур до передачі ходу (від 1 до 20).
Гравці ходять по черзі, намагаючись переграти суперника.
Перемагає той, хто набере більше очок до моменту, коли в обох закінчаться варіанти ходів.

3.Гра проти бота (Player vs IA)
Змагання з алгоритмом штучного інтелекту.
Ви робите хід першим.
Бот автоматично аналізує поле та обирає найвигіднішу позицію для своєї фігури (логіка вибору відображається в консолі).
Гра триває до повної відсутності ходів в обох сторін.
